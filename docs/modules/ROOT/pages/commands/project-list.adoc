= project list
List projects available for use with the 'boot new' and 'boot add' commands

== NAME
project list

== SYNOPSIS
====
[source]
----
project list --json boolean --help 
----
====

== OPTIONS
`--json` boolean JSON format output *Optional, default = false*

`--help or -h`  help for project list *Optional*
